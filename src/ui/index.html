Architecture
SSE Stream — Direct HTTP streaming pushes query metrics via /api/stream.
Dead Letter Queue — Corrupt events route to Kafka DLQ topic automatically.
Prometheus Exporter — Pipeline metrics served at /metrics endpoint.
Anomaly Tagging — Flags spill, queue bound, compile bound, and execution spikes in real-time.
